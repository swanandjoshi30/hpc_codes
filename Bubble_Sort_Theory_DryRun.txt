Parallel Bubble Sort (Odd-Even Transposition)
-----------------------------------------------
THEORY:
Standard Bubble Sort is inherently sequential. To parallelize it, we use the Odd-Even Transposition Sort:
1. The algorithm functions in phases.
2. Even Phase: Compare and swap pairs starting at even indices (0, 2, 4...).
3. Odd Phase: Compare and swap pairs starting at odd indices (1, 3, 5...).
4. Because pairs in each phase are independent, the comparisons can be done in parallel using '#pragma omp parallel for'.

DRY RUN:
Array: {5, 2, 9, 1}
1. Phase 0 (Even):
   - Pair(5, 2) -> Swap {2, 5}
   - Pair(9, 1) -> Swap {1, 9}
   - Array: {2, 5, 1, 9}
2. Phase 1 (Odd):
   - Pair(5, 1) -> Swap {1, 5}
   - Array: {2, 1, 5, 9}
3. Phase 2 (Even):
   - Pair(2, 1) -> Swap {1, 2}
   - Pair(5, 9) -> No swap
   - Array: {1, 2, 5, 9} (Sorted)
